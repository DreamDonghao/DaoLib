Detailed Description

RGBA颜色类

表示一个32位RGBA颜色值，支持整数通道(0-255)和浮点透明度(0.0-1.0)。 提供十六进制字符串转换、颜色操作和格式验证功能。

将十六进制颜色字符串转换为ColorRGBA对象

Parameters
hex十六进制颜色字符串，支持格式：RRGGBB、#RRGGBB、RRGGBBAA或#RRGGBBAA
Returns
转换后的ColorRGBA对象，如果格式无效则返回白色(#FFFFFF)

算法步骤：

  1. 移除可选的'#'前缀
  2. 验证长度必须是6(RGB)或8(RGBA)
  3. 将每两个字符转换为一个字节值(0-255)
  4. 检测转换错误：值为0但对应的十六进制字符不是"00"
